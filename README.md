Стартовый набор для выполнения проектной работы Место

Для запуска приложения необходимо использовать live-server
![изображение live-server](image.png)
Live-server является расширением VSCode
https://alex020160.github.io/mesto-production/ - ссылка на опубликованный проект
публичный репозиторий с опубликованным проектом (mesto-production): https://github.com/alex020160/mesto-production.git
Команда для запуска локального сервера: npm run dev
Команда для сборки проекта: npm run build
Команда для публикации проекта: npm run deploy
